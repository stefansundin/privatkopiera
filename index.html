Hur fungerar Privatkopiera?

Tillägget tar reda på vilka addresser du kan använda för att ladda ned videoklipp från ett antal Play-tjänster.

Privatkopiera kräver att du vet hur man använder FFmpeg (se Installera FFmpeg). Det är egentligen väldigt enkelt. Följ dessa steg:

  1. Hitta en video på svtplay.se som du vill ladda ned.
  2. Öppna Privatkopiera genom att klicka på VHS-ikonen till höger om addressfältet.
  3. Vänta några sekunder tills Privatkopiera hittar strömmen.
  4. Klicka på knappen Kopiera kommando.
  5. Klistra in kommandot i en terminal (t.ex. cmd.exe i Windows, använd högerklick för att klistra in). Tryck på enter-tangenten för att starta nedladdningen.
  6. Förhoppningsvis har du filen nedladdad efter några minuter.
  7. Obs: FFmpeg spottar gärna (och ofta) ur sig varningar. Om programmet fortsätter att köra så bör du ta det lungt, det finns en bra chans att det fungerar ändå. Ha tålamod och ge det en stund.

Senaste versionen stöder:

  • svtplay.se (även direktsändningar)
  • svt.se
  • sverigesradio.se
  • urplay.se
  • urskola.se
  • tv4.se
  • tv4play.se (ej premium/direktsändningar)
  • tv.nrk.no och radio.nrk.no
  • dr.dk

Obs: Nedladdning av undertexter stöds inte på alla sidor.

Installera FFmpeg

FFmpeg är ett program för som används för konvertering av videofiler, men det kan även ladda ned videoströmmar.

För tillfället rekommenderas att ladda ned FFmpeg från yt-dlp/FFmpeg-Builds på GitHub för att de har löst en bugg med undertexter från SVT. Det finns tyvärr bara tillgängligt till Windows och Linux.

Till macOS kan du ladda ned från https://evermeet.cx/ffmpeg/. Här kvarstår dock buggen med undertexter från SVT.

Hitta andra möjliga nedladdningar på ffmpeg.org.

Om kommandotolken har problem att hitta ffmpeg så kan du behöva konfigurera sökvägen i tilläggets inställningar.

Om det behövs så kan du modifiera kommandot som Privatkopiera ger dig. Se FFmpeg-dokumentationen.

Obs: FFmpeg spottar gärna (och ofta) ur sig varningar. Om programmet fortsätter att köra så bör du ta det lungt, det finns en bra chans att det fungerar ändå. Ha tålamod och ge det en stund.

Tips

Om du ändrar filändelsen till .mka så laddas bara ljudet ned. Likaväl kan du använda .srt för att bara ladda ned undertexter. Undertexter stöds ej på alla sidor!
